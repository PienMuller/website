If you’re just getting started with magical girl series and looking for a show to really get into the genre I’d recommend shows like Princess Tutu, purely because it only has 26 episodes so its not too much to start with and because it has no sideshows like Sailor Moon and Cardcaptor Sakura, which make it difficult to figure out the order to watch it.

Also I'd recommend all the pretty cure series, because (almost) every season has a unique story, so you can watch it in whatever order you like. Plus every once in a while they come out with a Pretty Cure All Stars, which includes all previous pretty cure characters. They still release a new season every year.
